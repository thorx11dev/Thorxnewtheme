## 3. THE PROBLEM & SOLUTION

### 3.1 The Problem (The Loophole)
Most advertisers run video ads on social media platforms such as YouTube, Instagram, Facebook, and TikTok. These ads pull minimal attention from viewers — resulting in fewer real leads for the advertiser, wasted ad spend, and passive, bot-driven impressions that deliver no real value.

Traditional digital advertising suffers from:
- **Budget leakage** — caused by passive, ignored impressions
- **Bot-driven fraud** — fake clicks and views with no real human attention
- **Poor conversion rates** — low engagement translating to low leads

### 3.2 The Thorx Solution
Thorx fills this gap. On Thorx, users don't just passively view ads — they actively engage with advertiser content and help generate real, verified leads through Thorx's proprietary **LeadX system**.

Thorx implements a **dual-phase verification layer** that bridges passive viewing and active engagement — ensuring better conversion rates for advertisers while rewarding users for verified, proven attention.

## 7. CORE ENGINES

---

### ENGINE A: The Attention Marketplace

**Purpose:** The core USP — maximizing ad memorability and conversion through a structured, two-part verified attention flow.

#### Phase 1 — The Video Player
- Users view a **15–25 second video ad** delivered via a proprietary, pre-built **Waterfall Video Player**
- **Waterfall Model:** If Ad Network 1 fails to load or runs out of inventory, the player automatically falls to Network 2, then Network 3 — ensuring zero downtime and maximum fill rates
- **Targeting:** Upon account creation, users input their interest profiles. The waterfall system filters and serves relevant ads to optimize natural conversion

#### Phase 2 — The Sandbox Web Panel
- After the video ends, the user requests to proceed
- The advertiser's product landing page loads inside a **dedicated Thorx sandbox container** (iframe / secure sub-window)
- **The Super AI Attention Detector (Hidden):** The user must explore the page for 15 seconds. To prevent gaming, all tracking mechanics are invisible — the user only sees a premium minimalist countdown timer

**Hidden behavioral payload compiled by the AI Attention Detector:**
| Signal | Description |
|---|---|
| **Tab Visibility API** | Instantly pauses the countdown if user switches tabs or minimizes the window |
| **Micro-Movement Delta** | Detects cursor (x, y) coordinates or touch input to ensure the device is being actively used |
| **Scroll Vector** | Tracks if the user scrolls at least 10–20% of the viewport height, proving active reading/exploration |

**Payout Condition:** If real attention is verified across all signals → balance is updated. If behavioral thresholds aren't met → no payout issued.

---

### ENGINE B: AI-Driven CPA Offers

**Purpose:** A high-yield secondary revenue stream featuring task-based earning actions.

**Example offer types:**
- Downloading apps
- Leaving social media reviews
- Completing registrations
- Watching YouTube videos
- Following accounts

#### Curation Process
The Thorx team cherry-picks the most reliable, highest-converting offers from established Cost-Per-Action (CPA) networks.

#### Automated Proof Vetting (AI Agent)
Users upload image or video proof of completed tasks. A custom AI Agent processes submissions through a **multi-tier fraud check:**

| Tier | Check | What It Does |
|---|---|---|
| **Tier 1** | Metadata/Hash Extraction | Instantly drops duplicate images, recycled proofs, or files with mismatched creation timestamps |
| **Tier 2** | Advanced OCR + LLM Analysis | Extracts text from screenshots to verify target handles, specific comment text, and modern UI structures |
| **Tier 3** | Escrow Release | Approved tasks enter "Pending" escrow status before admin panel clears for final wallet release |

---

### ENGINE C: The 3-Division Referral Matrix

**Purpose:** Drive organic viral growth via a multi-layered referral infrastructure across three tiers (User A → User B → User C).

Commission structures are tied to **payout requests**, turning user acquisition into a self-sustaining ecosystem.

#### Referral Tier Structure

| Tier | Trigger | Who Gets Paid | Commission |
|---|---|---|---|
| **Level 1** | User B withdraws | User A receives | 15% of User B's gross earnings |
| **Level 2 (Part A)** | User C withdraws | User B receives | 15% of User C's gross earnings |
| **Level 2 (Part B)** | User C withdraws | User A receives | 7.5% of User C's gross earnings |
| **Solo Loop** | Organic user (no referral) withdraws | Platform (Admin) receives | 15% Admin Charge |

#### How Commissions Work
- Commissions are **deducted from the withdrawing user's gross earnings**, not added as a bonus from the platform
- This is built into the Net-First UI — users always see their net-after-deductions balance

## 9. FINANCIAL MECHANICS & WALLET LOGIC

### 9.1 Net-First UI Strategy
To prevent user churn caused by psychological friction from large payout deductions, Thorx implements a strict **Net-First UI Pattern:**

- Users **never** see an inflated gross balance that gets heavily slashed at checkout
- All wallet views, task rewards, and dashboards calculate fees **silently in the background**
- Displayed balance = exactly what the user can withdraw
- **Tagline: "What You See Is What You Get."**

### 9.2 Deduction Flow at Withdrawal

```
[Gross Ad Revenue Generated]
         │
         ├───► 1. Platform Fee (12% flat — system maintenance)
         │
         └───► 2. Network Distribution (based on user type)
                      │
                      ├───► If Referred: Level 1 Split (15%) + Level 2 Split (7.5%)
                      │
                      └───► If Unreferred (Solo): Flat Admin Charge (15%)
```

### 9.3 Total Deduction Matrix

| User Classification | Platform Fee | Network / Admin Cut | **Total Deducted** |
|---|---|---|---|
| Referred User (Base Level C) | 12% | 15% (Lvl 1) + 7.5% (Lvl 2) | **34.5%** |
| Organic / Solo User (no referral) | 12% | 15% (Admin Charge) | **27.0%** |

### GENERAL / ABOUT THORX

**Q: What is Thorx?**
A: Thorx is an AI-powered digital engagement platform that lets you earn real money by paying genuine attention to ads and completing online tasks — while helping businesses get verified, high-quality leads.

**Q: How is Thorx different from other earning sites?**
A: Most earning platforms have no real fraud prevention, pay tiny amounts, and slash your balance with hidden fees. Thorx uses a hidden AI attention detector to verify every ad view, an AI agent to vet every task submission, and a strict "What You See Is What You Get" wallet system — so there are no surprise deductions at withdrawal.

**Q: Who can join Thorx?**
A: Anyone in Pakistan can join Thorx as an Earner. Advertisers (businesses wanting real leads) can also join through the Advertiser Portal.

**Q: Is Thorx an app?**
A: Thorx is a web-first platform — you access it directly through your browser on mobile or desktop. No app download or installation is required.

**Q: Is Thorx legitimate?**
A: Yes. Thorx uses AI-powered systems to ensure fair, transparent earnings. Every ad view is verified by a real attention detection engine, and every task submission goes through a multi-tier AI fraud check before approval.

---

### EARNING / GETTING STARTED

**Q: How do I start earning on Thorx?**
A: Sign up, complete your profile with your interest categories, navigate to the Work Area, and start watching ads or completing offers. Your wallet updates as you earn.

**Q: What can I earn on Thorx?**
A: You can earn by:
- Watching video ads (Engine A — Attention Marketplace)
- Completing CPA offers like downloading apps, leaving reviews, or signing up for services (Engine B)
- Earning referral commissions when people you've invited withdraw earnings (Engine C)

**Q: How much can I earn?**
A: Earnings vary based on the number of ads available, the offers you complete, and your referral network. Thorx displays your exact net earnings in real time — what you see is exactly what you'll receive.

**Q: Do I need to keep my screen active while watching an ad?**
A: Yes. Thorx uses an AI attention detection system that monitors tab visibility, cursor movement, and scroll behavior. If you switch tabs, leave the page, or stop interacting, the timer pauses and no payout is issued until real attention is verified.

### WALLET & WITHDRAWALS

**Q: How does the wallet work?**
A: Your wallet always shows your net withdrawable balance — after all fees have already been calculated in the background. There are no surprise cuts at checkout. What you see is what you get.

**Q: What fees does Thorx charge?**
A: Thorx charges a flat 12% platform fee on all withdrawals for system maintenance. Additionally:
- If you joined via a referral link, a network distribution fee applies (paid to your referrers)
- If you joined organically (no referral), a flat 15% admin charge applies

**Q: What are the total deductions from my withdrawal?**
A:
- Referred users: 34.5% total deducted (12% platform fee + 15% Level 1 + 7.5% Level 2 referral shares)
- Solo/organic users (no referral): 27% total deducted (12% platform fee + 15% admin charge)

**Q: How do I withdraw my earnings?**
A: Go to the Wallet section in your User Portal and request a withdrawal. Payouts are sent directly to your JazzCash or EasyPaisa account.

**Q: How long does a withdrawal take?**
A: Processing times depend on your withdrawal request and internal review. You will be notified of your payout status in your portal.

**Q: Is there a minimum withdrawal amount?**
A: Minimum withdrawal limits are set and visible in your Wallet section.

### OFFERS / TASKS (ENGINE B)

**Q: How do CPA offers work?**
A: CPA (Cost-Per-Action) offers are tasks curated by the Thorx team. You complete the required action — like downloading an app or leaving a review — upload proof, and the AI agent verifies your submission.

**Q: What proof do I need to submit for offers?**
A: You submit a screenshot or video proving you completed the task. The AI agent checks for authenticity using metadata analysis, OCR text extraction, and behavioral pattern review.

**Q: How long does offer approval take?**
A: After submission, approved tasks enter a "Pending" escrow state. Final wallet release happens after admin panel review. You'll receive a notification when your balance is updated.

**Q: What happens if my proof is rejected?**
A: If the AI agent or admin detects duplicate, recycled, or fraudulent proof, the submission is rejected and no payout is issued. Repeated fraudulent submissions may result in account action.

---

### REFERRALS (ENGINE C)

**Q: How does the referral system work?**
A: Thorx uses a 3-tier referral structure. When you invite someone (User B) and they withdraw earnings, you receive 15% of their gross withdrawal. If User B invites someone (User C) who withdraws, you receive 7.5% and User B receives 15%.

**Q: Are referral commissions added on top of my earnings?**
A: No. Referral commissions are deducted from the withdrawing user's gross earnings and distributed to their referrers. They are not an additional platform cost.

**Q: What is the Admin Charge for solo users?**
A: If someone joins Thorx without a referral link, a 15% admin charge is applied to their withdrawals. This replaces the referral network distribution that would otherwise apply.

**Q: How do I refer someone?**
A: Go to your Referral Dashboard in the User Portal to find your unique referral link or code. Share it with friends and family.

### AD-BLOCKER & TECHNICAL

**Q: Why is Thorx asking me to disable my ad-blocker?**
A: Thorx delivers real ad content as the core earning mechanism. Ad-blockers prevent ads from loading, which means the earning loop cannot function. You must whitelist Thorx in your ad-blocker to access the earning areas.

**Q: Does Thorx work on mobile?**
A: Yes. Thorx is designed mobile-first and works on all modern mobile browsers. No app download needed.

**Q: Is my data safe on Thorx?**
A: Thorx uses secure authentication, encrypted data handling, and AI-powered fraud systems to protect user data and prevent abuse.

### FOR ADVERTISERS

**Q: What is the LeadX system?**
A: LeadX is Thorx's proprietary lead generation system for advertisers. It combines the Attention Marketplace and CPA offer infrastructure to deliver verified, high-quality leads — not just impressions. Advertisers can opt into LeadX or launch their own direct campaigns.

**Q: Do I have to use LeadX as an advertiser?**
A: No. LeadX is optional. You can use it for best results or choose to launch your own ad campaigns or offers directly through the Advertiser Portal.

**Q: What kind of engagement do I get as an advertiser?**
A: Every ad view on Thorx is verified through a dual-phase system: users watch your ad fully, then actively explore your landing page in a sandbox environment. AI behavioral tracking confirms real attention before any payout is made to the user.
