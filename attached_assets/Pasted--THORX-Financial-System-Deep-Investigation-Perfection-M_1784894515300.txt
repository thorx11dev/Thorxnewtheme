# Primary Objective

Verify that THORX behaves exactly like this:

When a user earns money:

• The real PKR amount is permanently stored.

• TX-Points are generated only for engagement and display.

• The real PKR value never changes afterwards.

• TX-Points can change in future without affecting historical earnings.

During withdrawal:

The system must NEVER calculate:

Points × Current Conversion Rate

Instead it must always use:

Historical Real PKR Values

stored during earning.

Changing conversion rates tomorrow must never increase or decrease ThorX profit.

Historical earnings must remain financially identical forever.

This rule is one of the most important financial principles of THORX.

If this rule is violated anywhere in the codebase, fix it.
