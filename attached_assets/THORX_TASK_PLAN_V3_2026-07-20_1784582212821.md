### TASK 2 — Add UNIQUE Constraints for identity + phone (Finding 1-B)
**Priority:** HIGH | **Files:** `shared/schema.ts` + raw SQL

**Fix in schema.ts:**
```typescript

phone: text("phone").notNull(),  // partial unique via raw SQL below
```

**Raw SQL to apply (non-TTY env):**
```sql
-- Partial unique: only enforce uniqueness for non-empty values
-- (many users registered with phone="" before this constraint)
CREATE UNIQUE INDEX IF NOT EXISTS users_identity_unique 
  ON users(identity) WHERE identity != '' AND identity IS NOT NULL;
CREATE UNIQUE INDEX IF NOT EXISTS users_phone_unique 
  ON users(phone) WHERE phone != '' AND phone IS NOT NULL;
```

**Done looks like:**
- Attempting to register two accounts with the same non-empty identity → second rejected at DB level
