# THORX — Final Implementation & Investigation Directive for Replit Agent

## Purpose

Yeh document THORX ke Replit Agent ke liye ek official implementation directive hai. Iska maqsad yeh ensure karna hai ke THORX ka har system bilkul waisa hi kaam kare jaisa founder expect karta hai.

THORX sirf Pakistani users ke liye design kiya gaya hai. Isliye tamam systems, rewards, economy, fraud detection aur monetization ko Pakistan-centric approach ke saath implement aur optimize kiya jaye.

## Replit Agent Ke Liye Sabse Important Instruction

Tumhein superficial review nahi karni.

Tumhein:

* Super-deep code investigation karni hai.
* Line-by-line verification karni hai.
* Har engine ka actual runtime behavior test karna hai.
* Har API endpoint verify karna hai.
* Har database write operation inspect karni hai.
* Har transaction aur reward flow validate karna hai.
* Har edge case test karna hai.

* Har expectation mismatch detect karna hai.

Agar koi issue mile:
