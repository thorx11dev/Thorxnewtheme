Engine B: AI-Driven CPA Offers
Engine B serves as a high-yield secondary revenue stream featuring actions like downloading apps, leaving social media reviews, or completing registrations.

Curation: The Thorx team cherry-picks the most reliable, highest-converting offers from established Cost-Per-Action (CPA) networks.
Automated Proof Vetting: Users upload image/video proof of completed tasks. Instead of manual bottlenecks, a custom AI Agent processes submissions through a multi-tier fraud check:
Metadata/Hash Extraction: Instantly drops duplicate images, recycled proofs, or files with mismatched creation timestamps.
Advanced OCR + LLM Analysis: Extracts text from the screenshot to verify target handles, specific comment text, and modern UI structures.
Escrow Release: Approved tasks enter a "Pending" escrow status before being cleared by the admin panel for final wallet release.
Engine C: The 3-Division Referral Matrix
Engine C drives organic viral growth via a multi-layered referral infrastructure spanning three tiers (User A → User B → User C). Commission structures are tied to payout requests, turning user acquisition into a self-sustaining ecosystem.

Tier 1 Referral (Level 1): When User B initiates a withdrawal, a 15% referral commission is deducted from their gross earnings and distributed to User A.
Tier 2 Referral (Level 2): When User C requests a payout, User B receives a 15% commission, and User A receives a 7.5% commission from User C's gross earnings.
The Solo Loop / Admin Charge: Any user who joins the platform organically, without a referral link or code, is subject to a flat 15% Admin Charge upon withdrawal.
